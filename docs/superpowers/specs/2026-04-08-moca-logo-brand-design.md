# MOCA Logo & Brand Icon — Design Spec
**Date:** 2026-04-08  
**Approach:** Approach 2 — Logo + Full PWA Icon Set

---

## Overview

Replace the current purple lightning bolt favicon and plain-text Navbar header with a fully branded MOCA identity in mocha-latte tones. Deliver a reusable `Logo.jsx` component and a complete PWA icon set.

## Files Changed / Created

### 1. `mass-market-app/public/favicon.svg` — Updated
Same lightning bolt SVG shape as today. Colors replaced:
- `#863bff` / `#7e14ff` → `#5c3317` (bolt fill)
- Internal glow ellipses → warm cream/brown tones
- Background: transparent (renders on white/cream Navbar)

### 2. `mass-market-app/src/components/Logo.jsx` — New
Reusable component. Props:

| Prop | Type | Default | Values |
|------|------|---------|--------|
| `size` | string | `"md"` | `"xs"` / `"sm"` / `"md"` |

Rendered output per size:

| Size | Bolt | Wordmark | Subtext |
|------|------|----------|---------|
| `xs` | 16px | — | — |
| `sm` | 20px | "MOCA" bold | — |
| `md` | 28px | "MOCA" bold | "by Alon Yochelman" |

Colors: bolt `#5c3317`, wordmark `#3b1f0d`, subtext `#8a6a4a`. Font: system sans-serif, weight 900 for wordmark, 400 for subtext.

### 3. `mass-market-app/src/components/Navbar.jsx` — Updated
Replace the two `<span>` text elements in the logo `<NavLink>` with `<Logo size="md" />`. No other Navbar changes.

### 4. `mass-market-app/public/icons/icon-192.png` — New
- Size: 192×192px
- Shape: Rounded square (border-radius ~22% of size ≈ 42px)
- Background: gradient `#f5ede0` → `#e8d5bc` (top-left → bottom-right)
- Content: lightning bolt centered, fill `#5c3317`, scaled to ~60% of canvas
- Generated by: `anthropic-skills:canvas-design`

### 5. `mass-market-app/public/icons/icon-512.png` — New
Same as icon-192.png at 512×512px. Used for PWA splash screen and app store listings.

### 6. `mass-market-app/public/icons/icon-180.png` — New
Same design at 180×180px. Referenced as Apple Touch Icon (`<link rel="apple-touch-icon">`).

### 7. `mass-market-app/public/manifest.json` — Updated
```json
{
  "name": "MOCA — Mobile Operators Competitor Analysis",
  "short_name": "MOCA",
  "icons": [
    { "src": "/icons/icon-192.png", "sizes": "192x192", "type": "image/png", "purpose": "any maskable" },
    { "src": "/icons/icon-512.png", "sizes": "512x512", "type": "image/png", "purpose": "any maskable" }
  ]
}
```

### 8. `mass-market-app/index.html` — Updated
Add/update:
```html
<link rel="icon" type="image/svg+xml" href="/favicon.svg" />
<link rel="apple-touch-icon" href="/icons/icon-180.png" />
<link rel="manifest" href="/manifest.json" />
```
